Análisis exploratorio de Ventas Online Retail. 
Este proyecto corresponde al Trabajo Fin de Máster y consiste en un análisis exploratorio de datos (EDA) sobre un dataset de ventas de comercio electrónico, integrando información transaccional y datos de población para enriquecer el análisis.

El objetivo principal ha sido entender el comportamiento de las ventas, detectar patrones relevantes y extraer conclusiones que puedan aportar valor desde una perspectiva analítica y de negocio.

Objetivos del proyecto.
	•	Limpieza y preparación de datos.
	•	Análisis exploratorio de ventas.
	•	Integración de datasets adicionales (población).
	•	Cálculo de métricas como facturación total y facturación per cápita.
	•	Visualización de resultados.
	•	Desarrollo de un dashboard en Power BI.


Tecnologías utilizadas.
	•	Python
	•	Pandas
	•	Visualización de datos (matplotlib / seaborn)
	•	Power BI
	•	Git & GitHub


Estructura del repositorio.
	•	data/ → Datos originales y procesados.
	•	notebooks/ → Scripts de limpieza, análisis y visualización.
	•	figures/ → Gráficos generados.
	•	dashboards/ → Archivo Power BI (.pbix).
	•	Informe/ → Informe final en PDF con el análisis completo.


Resultados principales.

El análisis permitió identificar:
	•	Los países con mayor volumen de facturación.
	•	Diferencias entre facturación total y facturación ajustada por población.
	•	Patrones de comportamiento en las ventas.
	•	Insights relevantes para la toma de decisiones en entornos e-commerce.

El informe completo puede consultarse en la carpeta Informe/.


Conclusión.
Este proyecto demuestra el proceso completo de un análisis de datos: desde la limpieza y transformación hasta la visualización y comunicación de resultados mediante un dashboard y un informe final.
